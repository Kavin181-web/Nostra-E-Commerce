# Nostra - Modern E-Commerce Website

This project is a clean and professional e-commerce storefront created using HTML, CSS, and JavaScript. The main purpose of this task is to practice building a multi-page responsive website with interactive features.

In this project I created a homepage with an image slider a collections page with search and category filtering and a contact page with form validation!

## Features
- Interactive hero slider for featured collections
- Product search functionality on the collections page
- Category-based filtering (Clothing, Footwear, Accessories)
- Fully responsive design for mobile and desktop
- Contact form with real-time validation
- Professional multi-column footer with quick links

## Technologies Used
- HTML
- CSS (Flexbox & Grid)
- JavaScript
